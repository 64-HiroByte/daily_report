## わかったこと
- React Router v5 と v7 の違い
    - `react-router-dom` が不要になり `react-router` に統一された
    - `Switch` が `Routes` に変わった
    - `Route` は `element={<Component />}` と書く必要がある
    - `useHistory` が廃止され `useNavigate` に置き換わった
    - `to={{ pathname, state }}` が `to + state` に分離された  
- Atomic Design について
