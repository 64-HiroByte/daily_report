## わかったこと
- データベースの初期化のみを実行する場合、全てのコンテナを起動するのではなく、`--wait` を使って必要なコンテナだけを起動すれば良い。

```sh
$ docker compose -f docker-compose.yml -f docker-compose.test.yml up -d --wait db
```
