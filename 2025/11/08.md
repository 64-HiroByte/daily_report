## わかったこと
- Tailwind CSSを使って条件分岐させてスタイリングを適用するのに`tailwind-merge` , `clsx` の2つのパッケージを使用すると便利
- これらを使う際には、`src/lib/utils.ts` を作成して、次のように記述するのがお作法
```ts
import clsx from "clsx";
import { twMerge } from "tailwind-merge";
import type { ClassValue } from "clsx";

export const cn = (...inputs: ClassValue[]) => {
  return twMerge(clsx(inputs));
};

```            
- `Object.entries(obj)` でオブジェクトを `[key, value]` の配列に変換できる。そして、`Object.fromEntries(...)` で再びオブジェクトに戻すことができる。この仕組みを使うと、フォームのフィールドの値について `trim()` といった前処理するのに便利

 ```ts
// 例
const trimmed = Object.fromEntries(
  Object.entries(newUserData).map(([k, v]) => [k, (v as string).trim()])
) as UserFormType;
```
