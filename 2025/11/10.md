## わかったこと
- 配列をループしながらの要素の加工にはmap()とflatMap()を使う。
> flatMap() は Array インスタンスのメソッドで、最初にマッピング関数を使用してそれぞれの要素をマップした後、結果を新しい配列内に平坦化します。これは、 map() の後に深さ 1 の flat() を行うのと同じですが (arr.map(...args).flat())、これら 2 つのメソッドを別々に呼び出すよりもわずかに効率的です。
[引用元: MDN  - Array.prototype.flatMap() より](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Global_Objects/Array/flatMap)
```js
// --- 使用例 ---

srcs = ["Hello world", "nice to meet you"];

// 旧: ループを使って文を分割し3文字以上の単語の配列を作成する
var result = []
for (var i = 0; i < srcs.length; i++) {
    var words = srcs[i].split(" ")
    for (var j = 0; j < words.length; j++) {
        if (words[j].length > 2) {
            result.push(words[j]);
        }
    }
}

// 新: flatMapを利用
const result = srcs.flatMap(src => {
    return src.split(" ").filter(w => w.length > 2);
})
```
