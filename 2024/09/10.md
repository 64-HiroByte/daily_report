## わかったこと
- psycopg3を使うなら`RUN apt-get install -y libpq-dev`は不要そう
- `RUN apt-get install -y build-essential`も必要ではなく、結果的に`RUN apt-get update`も不要
- `platform: linux/amd64`はイメージのプラットフォームを指定している
